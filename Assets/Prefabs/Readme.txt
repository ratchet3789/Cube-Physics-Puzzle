HUD: Canvas with Win Message and Button on it


DestroyableCube: The default cube that can be left clicked and destroyed. Contains TAG - Destroyable.

NonDestroyableCube: A Non-destroyable cube...

Objective: The Objective the Player Cube must hit

PlayerCamera: The Players Camera, containing refresh script.

PlayerCube: The Player Cube. Contains TAG - Player.

Creating a Level:
Throw the HUD, PlayerCamera, PlayerCube and Objective
in the scene. Open the PlayerCamera and set the ResetButton
to the Button in Scene.
Then open then Objective and set the Win Message to
Winner Text.
